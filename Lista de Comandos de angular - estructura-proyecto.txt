Lista de Comandos de angular - estructura

0. crear un nuevo proyecto de angular
ng new proyecto-angular

1. generar un componente (inicio)
ng g c inicio

2. generar un módulo (producto)
ng g m producto --routing

3. Generar un componente (productos y producto-detalle) parte del módulo producto
ng g c producto/components/productos
ng g c producto/components/producto-detalle

4. generar un módulo (auth)
ng g m auth --routing

5. Generar los componentes (login y registro) parte del módulo producto
ng g c auth/components/login
ng g c auth/components/registro

6. generar un módulo (admin)
ng g m admin --routing

7. Generamos el componente (layout)
ng g c layout
